How to create a new paper

  1. go to the papers page
  2. click on the "add a new paper" link
  3. fill out the form
  4. click submit

How to create a new dataset

  1. go to the papers page and find the paper you want the dataset to be associated with and click on the ID to go to the paper's profile page
  2. alternatively if you know the paper's id, e.g. NNN, then go to the paper's profile page: http://cistrome.org/newdc/paper_profile/NNN/ where NNN is the paper's id
  3. click on the "add a new dataset" link
  4. fill out the form
  5. click submit

How to create a new sample

  1. go to the papers page and find the paper you want the sample to be associated and click on the ID to go to the paper's profile page
  2. alternatively if you know the paper's id, e.g. NNN, then go to the paper's profile page: http://cistrome.org/newdc/paper_profile/NNN/ where NNN is the paper's id
  3. click on the "add a new sample" link
  4. for each dataset in the sample, click on "add a dataset" and select the dataset. For example: GSM12345 is the sample treatment and GSM12346 is the sample control, then click "add a dataset" twice and select GSM12345 in the first menu and GSM12346 in the second menu. You can only select from the datasets that are associated with the paper.
  5. click submit
